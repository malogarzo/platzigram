<h1>Platzigram</h1>

<h4>Install</h4>
<ul>
	<li>$ git clone https://github.com/malogarzo/platzigram.git</li>
	<li>$ cd platzigram</li>
	<li>$ npm install</li>
</ul>

<h4>Run</h4>
<ul>
	<li>$ npm start</li>
</ul>
